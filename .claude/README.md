# Claude Code Working Directory

This directory contains ephemeral files created during Claude Code sessions.

## Structure

- `scratch/` - Temporary experiments, code snippets, rough drafts
- `notes/` - Session-specific notes and observations
- `questions.txt` - Working list of questions and todos

## Important

- Files here are **not committed** to version control (see `.gitignore`)
- For permanent documentation, use the `CLAUDE.md` files or `docs/` directory
- This directory is safe to clean up periodically
